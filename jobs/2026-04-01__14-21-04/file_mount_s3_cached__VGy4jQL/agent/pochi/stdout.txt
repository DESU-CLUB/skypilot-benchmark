  /data:

    --output /tmp/results.json

**Key design decisions:**

| Field | Value | Reason |
|---|---|---|
| `accelerators: A100:4` | 4× A100 | Matches the 4-GPU requirement |
| `use_spot: true` | Spot instance | Cost-efficient for batch workloads |
| `/data` → `MOUNT_CACHED` | FUSE mount with local SSD cache | Best for large datasets read repeatedly — avoids re-downloading data on every access |
| `/checkpoint` → `COPY` | Full download at job start | Model weights are read once; `COPY` avoids FUSE overhead and gives fastest repeated reads |
